# Simple Crowdfunding DApp (Skeleton)

Simple crowdfunding DApp for Blockchain 1 final exam.

## Structure
- contracts/ - Solidity contracts
- scripts/ - deployment scripts (placeholder)
- test/ - tests (placeholder)
- frontend/ - client app (placeholder)
- docs/ - documentation (placeholder)
- presentation/ - slides (placeholder)

## Contracts
- contracts/RewardToken.sol - custom ERC-20 reward token
- contracts/Crowdfunding.sol - crowdfunding logic + reward minting

## Quickstart (Local)
1. `cd /Users/test/tomi_archive/crowdfunding-dapp`
2. `npm install`
3. `npx hardhat node`
4. In a new terminal: `npm run deploy:local`
5. Paste addresses into `/Users/test/tomi_archive/crowdfunding-dapp/frontend/src/app.js`
6. Open `/Users/test/tomi_archive/crowdfunding-dapp/frontend/public/index.html`
